Linux运维4:磁盘挂载,包管理工具

mount磁盘挂载,umount磁盘卸载

yum包管理

磁盘挂载

  • 挂载:文件系统外通过关联至根文件系统上的某个目录来实现访问该磁盘
  • 挂载点:mount_point,用于作为另一个文件系统的访问入口

永久挂载:

  1. 接入物理磁盘
  2. 对物理磁盘进行分区(fdisk,lvm)见linux运维5
  3. 格式化磁盘分区(format)
  4. mount文件系统到指定挂载点
  5. vim /etc/fstab将磁盘写入文件,永久挂载
  6. reboot操作系统进行验证

umount [mount_point]卸载挂载的文件系统

yum包管理

配置本地yum镜像文件

yum常用命令

命令 作用
安装
yum install 全部安装
yum install {package} 安装指定安装包
yum groupinstall {group} 安装指定程序组
查找和显示
yum list 显示所有已经安装和可以安装的程序包
yum info {package} 显示安装包信息package
yum list {package} 显示指定程序包安装情况
yum grouplist 显示所有程序组信息
yum groupinfo {group} 显示程序组group信息
yum search string 根据关键字string查找安装包
卸载
yum remove {package} 删除程序包
yum groupremove {group} 删除程序组
