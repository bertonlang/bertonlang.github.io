用户和组命令 ：groupadd,groupmod,groupdel,useradd,usermod,userdel

文件目录权限 ：chmod,chgrp,chown

用户和组

groupadd ：建立新用户组

  • -g ：指定新建工作组的gid groupadd -g 2534 berton

groupmod ：修改用户组

  • -g ：为用户指定新的组标识号 groupmod -g 3534 berton

groupdel ：删除用户组

  • 删除指定用户组 groupdel berton

useradd ：添加用户

  • -u <用户id>：指定用户id
  • -d <登入⽬录>：指定⽤户的宿主目录(登入时的起始目录)
  • -g <群组>：指定⽤户所属的主群组
  • -G <群组>：指定用户所属的附加群组
  • -m：⾃动建⽴⽤户的登⼊目录（默认）
  • -M：不要⾃动建⽴⽤户的登⼊目录
  • -n：取消建⽴以⽤户名称为名的群组 useradd -u 2534 -d /home/berton/ -g berton -G root berton

usermod ：修改用户

  • -l <帐号名称> ：修改用户帐号名称
  • -d <登入⽬录> ：修改用户登入时的目录
  • -g <群组> ：修改用户所属的群组
  • -G <群组> ：修改用户所属的附加群组
  • -L ：锁定用户密码，使密码无效
  • -U ：解除密码锁定
  • -u <用户id> ：修改⽤户IDusermod -l xiaoming -d /home/xiaoming -g xiaoming berton

userdel ：删除用户

  • -f：强制删除用户，即使用户当前已登录
  • -r：删除用户的同时，删除与用户相关的所有文件userdel -rf xiaoming

su - ：切换用户

  • 切换到root用户 ：su - root

文件及目录权限

chmod ：修改权限

  • chmod 755 file_a：赋予file_a权限rwxr-xr-x
  • chmod u=rwx , g=rx , o=rx abc ：同上u=⽤户权限，g=组权限，o=不同组其他用户权限
  • chmod u-x , g+w abc：给abc去除⽤户执行的权限，增加组写的权限
  • chmod a+r abc：给所有⽤户添加读的权限 all
  • 目录 需要x（执行）权限进⾏打开。

chown ：修改拥有者

  • chown userB abc：改变abc的所有者为userB
  • chown root ./abc：改变abc这个目录的所有者是root
  • chown -R root ./abc

chgrp ：修改拥有组

  • chgrp it abc ：改变abc所属的组为it组
  • chown userB ：groupB abc ：改变abc的拥有者为userB及所属组groupB
  • chown -R userB ：groupB /test ：改变该目录下所有目录寄文件的拥有者及所属组

常用环境变量

  • PATH ：决定了shell将到哪些目录中寻找命令或程序 !!!!!!!!
  • HOME ：当前用户主目录
  • HISTSIZE ：历史记录数
  • LOGNAME ：当前用户的登录名
  • HOSTNAME ：指主机的名称
  • SHELL ：当前用户Shell类型
  • LANGUGE ：语言相关的环境变量，多语言可以修改此环境变量
  • PS1 ：基本提示符，对于root⽤户是#，对于普通用户是$

终端命令提示符设置（PS1）

  • 显示的是全路径 export PS1=“[\u@\h \w] #”
    • 大写的W是简介模式
    • 小写的w为全路径模式
    • root 为 #
    • 普通用户为 $

常用符号意义

  • \h：主机名简称

  • \H：完整的主机名称

  • \d ：代表⽇期，格式为weekday month date，例例如：”Mon Aug1”

  • \t ：显示时间为24小时格式，如：HH：MM：SS

  • \T ：显示时间为12小时格式

  • \A ：显示时间为24小时格式：HH：MM

  • \u ：当用户的账号名称

  • \v ：BASH的版本信息

  • \w ：完整的工作目录名称。绝对路径。家目录会以 ~代替

  • \W ：利⽤basename取得⼯作目录名称，所以只会列出最后⼀个目录
